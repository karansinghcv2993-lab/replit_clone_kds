# MyWave AI — Agent Catalogue

A marketing/catalogue site showcasing 352+ enterprise AI agents built for finance, supply chain, insurance, banking, healthcare, and SAP Business One.

## Stack

- **Runtime:** Bun 1.3
- **Framework:** TanStack Start (SSR React) + TanStack Router (file-based routing)
- **Styling:** Tailwind CSS v4 + shadcn/ui components
- **Build:** Vite 8 via `@lovable.dev/vite-tanstack-config`

## Running the app

```bash
bun run dev   # starts dev server on port 5000
```

The workflow **"Start application"** (`bun run dev`) is configured and runs automatically. The app is served at port 5000.

## Project structure

```
src/
  routes/       # File-based routes (TanStack Router)
    __root.tsx  # App shell — wraps every page
    index.tsx   # Home page (/)
  components/
    site/       # Page-level sections (HeroSection, CatalogueSection, etc.)
    ui/         # shadcn/ui primitives
  data/         # Static agent catalogue data
  lib/          # Utilities and error helpers
  styles.css    # Global Tailwind styles
```

## Notes

- `routeTree.gen.ts` is auto-generated by TanStack Router — do not edit by hand.
- The vite config is thin; most Vite/Nitro/Tailwind plugins are bundled inside `@lovable.dev/vite-tanstack-config`. Pass extra config via `defineConfig({ vite: { ... } })`.
- This project originated from Lovable — avoid force-pushing or rewriting published git history.
